ProWeb Kartu Stok (Stock Card)

Print Kartu Stok (Stock Card) of some Product / Product Variants on spesific Warehouse location

Display Lots/Serial Number and Expired Date

How to use:

1. Choose some Products / Product Variants

Go to Products / Product Variants menu then select some.

2. Click Action Button

Click Action Button then click Print Kartu Stok.

3. Choose Warehouse

Choose Warehouse location, then click Print Report button.

4. Kartu Stok (Stock Card) Report

Kartu Stok (Stock Card) Report will appear in PDF format file.
